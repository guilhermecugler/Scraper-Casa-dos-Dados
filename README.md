# 🚚 **Projeto Migrado!**
👉 O projeto original **Scraper Casa dos Dados** foi descontinuado.  
Agora, devido às **limitações impostas pelo site Casa dos Dados**, que restringem a quantidade de resultados retornados e dificultam a extração eficiente de dados, a nova abordagem utiliza **dados baixados diretamente do portal oficial do governo**, permitindo acesso completo e offline.

🔗 **Novo repositório:** [CasaDosDadosOffline](https://github.com/guilhermecugler/CasaDosDadosOffline)

> ⚠️ Recomendamos fortemente que utilize o novo projeto, pois este scraper não é mais eficaz devido às mudanças no site.

---

## ❗ Contexto da Migração

Devido a alterações recentes no **site Casa dos Dados**:
- O número de resultados por busca foi **limitado a apenas 200 registros**.
- Proteções adicionais (como a Cloudflare) dificultam ou impedem a coleta automática de dados em larga escala.
- Isso tornou inviável a manutenção de um scraper eficiente.

Por isso, o projeto foi **migrado para um novo repositório**, que realiza consultas **offline**, utilizando diretamente a **base de dados oficial disponibilizada pelo governo**.

🔗 Acesse o novo projeto: [CasaDosDadosOffline](https://github.com/guilhermecugler/CasaDosDadosOffline)

---

## 🛑 Status deste Repositório

Este repositório está **descontinuado** e será mantido apenas como referência histórica.  
Nenhum novo recurso ou correção será implementado aqui.

Para obter dados atualizados, utilize o [CasaDosDadosOffline](https://github.com/guilhermecugler/CasaDosDadosOffline).

---

## 📦 Sobre a Versão Antiga (Histórico)

O **Scraper Casa dos Dados** foi inicialmente desenvolvido para extrair dados do site usando **Playwright** e requisições assíncronas, conseguindo até 1000 resultados em aproximadamente 1 minuto.

No entanto, com as limitações impostas pelo site, a abordagem deixou de ser prática e eficiente.
